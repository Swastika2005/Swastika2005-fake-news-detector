 Fake News Detection System (Machine Learning + Streamlit)

This project is a Fake News Detection System built using Machine Learning (Logistic Regression), TF-IDF text vectorization, and a simple Streamlit web app.
It classifies news text as Real or Fake with high accuracy using natural language processing (NLP).

 Features

Detects whether a news article is Real or Fake

User-friendly Streamlit Web App

Supports:
⟫Typing text directly
⟫Uploading a .txt file

Machine Learning model trained using:
⟫TF-IDF Vectorizer
⟫logistic Regression
⟫Achieved ~98% Accuracy on test data

Tech Stack

>>Python
>>Scikit-Learn
>>Pandas / NumPy
>>Joblib
>>Streamlit
>>NLP (TF-IDF Vectorizer)

 Model Accuracy
